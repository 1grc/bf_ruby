# BfRuby

BfRuby is Brainf**k interpreter written in Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bf_ruby'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install bf_ruby

## Usage
```ruby
BfRuby::Translator.translate(brainf**k_source)
```
